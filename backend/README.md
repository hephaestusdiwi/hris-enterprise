# Backend

Laravel 12 API akan di-install di sini pada **STEP 11 — Laravel Installation**.

Struktur module (Modular Monolith) akan mengikuti:

```
app/Modules/{ModuleName}/
├── Controllers/
├── Models/
├── Services/
├── Repositories/
├── Policies/
├── Requests/
├── Resources/
├── Events/
├── Listeners/
├── Jobs/
├── Routes/
└── Tests/
```
